# VYNEDAM - Internship & Hackathon Platform

VYNEDAM is a platform connecting interns and hackathon participants with real projects, mentorship, and skill-building opportunities.

---

## 🚀 Features

- Skill-Based Internships  
- Real Projects  
- Mentorship  
- Certificates  
- Hackathon Registration & Payment Portal  
- Parallax tech background with 3D motion  
- Smooth animations with Framer Motion  

---

## 💻 Requirements

- Node.js (v16+ recommended)
- npm (comes with Node.js)

---

## 📦 Installation & Running the App

1. **Extract the ZIP file** to your local folder.  
2. **Open a terminal** in the project folder.  
3. **Install all dependencies** (React, React Router, Framer Motion, etc.):

```bash
`npm install`
`npm install react-router-dom`
`npm install framer-motion`
`npm start`
